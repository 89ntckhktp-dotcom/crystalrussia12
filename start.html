Начать играть

Следуйте простым шагам, чтобы начать играть в CRYSTAL RUSSIA прямо сейчас

1

Скачайте APK файл

Загрузите установочный файл игры на ваше Android-устройство. Файл полностью безопасен и проверен антивирусами.

Скачать APK (120 МБ)
2

Разрешите установку из неизвестных источников

В настройках безопасности вашего устройства разрешите установку приложений из неизвестных источников.

Путь: Настройки → Безопасность → Неизвестные источники (включить)

3

Установите приложение

Найдите скачанный файл в папке "Загрузки" и нажмите "Установить". Дождитесь завершения установки.

4

Запустите игру

После установки нажмите "Открыть" или найдите иконку CRYSTAL RUSSIA на рабочем столе.

Создайте аккаунт и начните своё приключение!

Частые вопросы

Безопасно ли?

Да, APK файл проверен антивирусами и не содержит вредоносного кода. Мы гарантируем безопасность ваших данных.

Обновления

Игра автоматически обновляется при запуске. Вам не нужно скачивать новые версии вручную.

Мультиплеер

Играйте с тысячами других игроков на одном сервере. Регистрация занимает менее минуты.

Поддержка

Если возникли проблемы с установкой, обратитесь в нашу техническую поддержку в соцсетях.
